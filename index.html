How to Play The Fishbowl

Phase 1: Setup & Data Collection

  1. **Create the Form:** Click the **"Go to Google Forms"** button on the main screen. Set up a simple form with one question using the **Paragraph (Long Answer)** format for players to write their messages.
  2. **Share & Collect:** Share the form link with all players to collect their submissions (e.g., inside jokes, personal compliments, confessions, etc.).
  3. **Download CSV:** Once all submissions are in, the admin goes to the **Responses** tab in Google Forms, clicks the Sheets icon, and then in the Google Sheet, selects **File > Download > Comma Separated Values (.csv)**.
  4. **Load Game:** Use the **"Upload Your CSV File"** section in this app to select and load the CSV file. The app will shuffle the messages and confirm when the game is ready.

Phase 2: Playing the Game (The Admin's Role)

The core gameplay is driven by the **Admin** (the person controlling this screen) who reads the messages to the group.

  1. **Start Reading:** The admin clicks the **"Read Next Message"** button to display the first message from the virtual "fishbowl."
  2. **The Reveal:** The admin reads the message aloud to the group.
  3. **The Guess/Reaction:** After the message is read, the **person for whom the message was written** gets a chance to guess who wrote it, or simply react to the statement.
  4. **Next Message:** Once the discussion for the current message is complete, the admin clicks **"Read Next Message"** again to continue the game.
  5. **End of Game:** The game continues until all messages have been read (the progress indicator shows $X/X$). Click **Reset Game** to shuffle and read the messages again if desired!
